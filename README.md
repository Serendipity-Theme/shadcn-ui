# Serendipity for shadcn/ui

Elegant, minimal, and clean color palette for your tools.

See other interfaces at the [official website](https://www.michaelandreuzza.com/vscode/serendipity/).

## Available themes

- **Midnight** — dark
- **Morning** — light
- **Sunset** — dark

## Installation

Copy a variant `globals.css` into your app and import in the root layout.

Available files: `midnight/globals.css`, `morning/globals.css`, `sunset/globals.css`, `tailwind.preset.js`.
